# Class diagrams
 
---

 ```mermaid
 classDiagram
     class 유저 {
         +사용자 인증 및 검증
         +사용자 정보 관리
         +사용자 상태 관리
         +보유 쿠폰 관리
         +주문 생성 요청(주문)
         +포인트 충전 요청(포인트)
     }
 
     class 포인트 {
         +포인트 잔액 관리
         +포인트 충전 처리(유저)
         +포인트 차감 처리(결제)
         +포인트 이력 관리
     }
 
     class 주문 {
         +주문 생성 및 검증(유저)
         +주문 상태 관리
         +주문 만료 관리
         +결제 요청 처리(결제)
         +쿠폰 적용 요청(쿠폰)
     }
 
     class 주문리스트 {
         +주문 상품 목록 관리
         +주문 상품 수량 관리
         +주문 상품 가격 관리
         +재고 확인 요청(재고)
     }
 
     class 상품 {
         +상품 정보 관리
         +상품 카테고리 관리
         +상품 가격 관리
         +상품 유효성 검증
     }
 
     class 재고 {
         +재고 수량 관리
         +재고 차감 처리(주문리스트)
         +재고 복원 처리(주문)
         +재고 유효성 검증
     }
 
     class 결제 {
         +결제 처리(주문)
         +결제 금액 검증
         +결제 이력 관리
         +쿠폰 할인 적용(쿠폰)
         +포인트 차감 요청(포인트)
     }
 
     class 쿠폰 {
         +쿠폰 정보 조회
         +쿠폰 발급 처리(유저)
         +쿠폰 만료 관리
         +쿠폰 사용 처리(주문)
         +쿠폰 유효성 검증
     }
 
     유저 "1" -- "1" 포인트 : 포인트 관리
     유저 "1" -- "0..*" 주문 : 주문 생성
     유저 "1" -- "0..*" 쿠폰 : 쿠폰 보유
     주문 "1" -- "1" 주문리스트 : 주문 상품 관리
     주문리스트 "1" -- "1..*" 상품 : 상품 정보 참조
     상품 "1" -- "1" 재고 : 재고 관리
     주문 "1" -- "1" 결제 : 결제 처리
     주문 "0..1" -- "0..1" 쿠폰 : 쿠폰 사용
 ```
